Key Components by Function
===========================

🔹 DATA PIPELINE
   └── src/data_preprocessing.py
       • Excel ingestion
       • Schema standardization
       • Feature engineering
       • Target variable creation

🔹 TEXT ANALYTICS
   └── src/text_analytics.py
       • Sentence embeddings (sentence-transformers)
       • KMeans clustering
       • Cluster labeling
       • Key phrase extraction

🔹 MACHINE LEARNING
   └── src/model_training.py
       • XGBoost/LightGBM training
       • Feature importance
       • Model evaluation (AUC, PR-AUC, Brier)
       • Model serialization

🔹 DEPLOYMENT
   ├── src/score.py                        # Scoring logic
   └── src/deploy_endpoint.py              # Azure ML deployment
       • Managed online endpoints
       • Blue/Green deployment
       • Traffic management

🔹 POWER BI INTEGRATION
   ├── powerbi/power_query_functions.m     # Batch scoring functions
   ├── powerbi/dax_measures.dax            # Business metrics
   └── powerbi/report_specification.md     # Dashboard design
       • 6 interactive pages
       • 40+ DAX measures
       • Row-level security

🔹 AUTOMATION
   ├── scripts/provision_azure_resources.*  # Infrastructure setup
   ├── scripts/test_endpoint.py            # Validation testing
   └── scripts/retrain_model.py            # Monthly retraining


Workflow
========

1. SETUP
   └── scripts/provision_azure_resources.ps1
       Creates Azure resources (ML workspace, compute, etc.)

2. TRAIN
   └── notebooks/end_to_end_pipeline.ipynb
       OR
   └── scripts/retrain_model.py --data data/warranty_dump.xlsx
       Trains model end-to-end

3. DEPLOY
   └── src/deploy_endpoint.py
       Deploys model to Azure ML endpoint

4. TEST
   └── scripts/test_endpoint.py
       Validates endpoint is working

5. POWER BI
   └── Use powerbi/ artifacts to build dashboard
       • Import Power Query functions
       • Add DAX measures
       • Build report pages

Dependencies
============

Python Packages (requirements.txt):
  • azure-ai-ml==1.12.0
  • pandas==2.1.4
  • scikit-learn==1.4.0
  • xgboost==2.0.3
  • lightgbm==4.2.0
  • sentence-transformers==2.3.1
  • joblib==1.3.2

Azure Resources:
  • Azure ML Workspace
  • Compute Cluster (STANDARD_DS3_V2)
  • Managed Online Endpoint
  • Key Vault
  • Application Insights

External Tools:
  • Power BI Desktop
  • Azure CLI
  • Python 3.9+


Quick Commands
==============

# Install dependencies
pip install -r requirements.txt

# Provision Azure
.\scripts\provision_azure_resources.ps1

# Train model
python scripts\retrain_model.py --data data\warranty_dump.xlsx

# Deploy endpoint
python src\deploy_endpoint.py

# Test endpoint
python scripts\test_endpoint.py

# Run notebook
jupyter notebook notebooks\end_to_end_pipeline.ipynb
